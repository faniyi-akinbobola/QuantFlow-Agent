# Evaluation Reports

This directory contains evaluation reports generated by the QuantFlow Agent evaluation framework.

## Report Types

### JSON Reports

- **Format**: `eval_report_YYYYMMDD_HHMMSS.json`
- **Content**: Complete evaluation results in JSON format
- **Use Case**: Programmatic analysis, CI/CD integration

### Markdown Reports

- **Format**: `eval_report_YYYYMMDD_HHMMSS.md`
- **Content**: Human-readable evaluation summary
- **Use Case**: Documentation, quick review

### HTML Reports

- **Format**: `eval_report_YYYYMMDD_HHMMSS.html`
- **Content**: Interactive HTML dashboard
- **Use Case**: Visual analysis, presentations

### Comparison Reports

- **Format**: `comparison_report_YYYYMMDD_HHMMSS.md`
- **Content**: Side-by-side comparison of multiple evaluation runs
- **Use Case**: A/B testing, performance tracking

## Generated Files

Reports are automatically generated when running:

```bash
# Local evaluation
python evals/runners/run_local.py

# CI/CD evaluation
python evals/runners/run_ci.py
```

## Report Contents

Each report includes:

- **Summary**: Pass rate, total tests, duration, cost
- **Metrics**: Latency statistics, usage metrics
- **Dataset Results**: Performance by dataset
- **Evaluator Scores**: Correctness, faithfulness, reasoning quality, etc.
- **Threshold Comparison**: Pass/fail status for each threshold
- **Failures**: Detailed failure examples (if enabled)

## Configuration

Report settings can be configured in `evals/configs/eval_config.yaml`:

```yaml
reporting:
  output_dir: "evals/reports"
  format: ["json", "html", "markdown"]
  include_failures: true
  include_examples: true
  max_examples_per_category: 5
  generate_charts: true
```
